Explanation of Attributes:

InnerSize: The width and height of the browser's viewport (the visible area inside the browser window).

OuterSize: The width and height of the browser window, including toolbars, scrollbars, and borders.

ClientSize: The width and height of the content area within the `` tag, excluding scrollbars.

BodySize: The full width and height of the `` tag, including padding and borders.

ScreenSize: The total width and height of the user's screen in pixels.

Screen Available Size: The width and height of the screen available for the browser, excluding UI elements like the taskbar.

Document Size: The width and height of the viewport for the `` element, excluding scrollbars.

Scroll Size: The total width and height of the document, including content outside the viewport that can be scrolled to.

Device Pixel Ratio: The ratio of the display's physical pixels to the CSS pixels, which determines the resolution of the screen.

Window Position: The X and Y coordinates of the browser window relative to the screen.
